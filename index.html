Welcome to betools.

A small set of tools I've created to make Python programming a bit nicer. The main one is CmdLine, a decorator that automatically creates command-line options using argparse. Here's how you use it:

from betools import CmdLine

@CmdLine("t", "option1")
def test1():
    """
    Description of option1
    """
    print("option1")

@CmdLine("u", "option2")
def test2():
    """
    Description of option2
    """
    print("option2")

@CmdLine("v", "option3")
def test3():
    """
    Description of option3
    """
    print("option3")


if __name__ == '__main__': CmdLine.run()
